What is the document object model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, allowing programs and scripts (like JavaScript) to interact with, access, and manipulate the content, structure, and styling of a web page dynamically.

You can select elements from the DOM using various methods in JavaScript. These methods allow you to target specific elements on a webpage based on their tag names, classes, IDs, attributes, and other properties.

2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

What is the event delegation in the context of the DOM why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of adding event listeners to each individual child element. When an event occurs on one of the child elements, the event is "delegated" to the parent, which handles it. This works because of event bubbling, where events propagate upward through the DOM tree from the event target to the parent elements.
